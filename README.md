# Express + Vite (React) + Vercel — Clean Starter
## Usage
1. Unzip and enter project folder.
2. Install dependencies:
   - root: `npm install`
   - client: `cd client && npm install`
3. Local dev:
   - `cd client && npm run dev` (frontend)
   - `node api/index.js` (backend) — for local combined testing, see notes.
4. To deploy on Vercel:
   - `vercel` (or via dashboard). The included vercel.json is configured for single-project deploy.
